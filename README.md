# Super Green Landscaping

A conversion-focused, responsive landscaping website built as a static site for GitHub and Cloudflare Pages.

## Preview locally

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080`.

## Before publishing

1. Replace the placeholder phone number, address, domain and service areas.
2. Replace sample reviews and rating with verified customer feedback.
3. Confirm offered services, property types, maintenance frequency and project capabilities.
4. Add verified licensing, insurance, certifications, warranties and years in business.
5. Connect the estimate form to a CRM or form service.
6. Add original project photography with descriptive alt text.
7. Create location and individual service pages as verified business details become available.

## Cloudflare Pages

Use no framework preset, no build command, and `/` as the output directory. Connect the GitHub repository for automatic deployments on each push.
